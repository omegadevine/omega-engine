# Omega Engine - Examples

This directory contains example code demonstrating various features of the Omega Engine.

## Python Examples

### Advanced Demo
`advanced_demo.py` - Comprehensive demonstration of physics, rendering, and effects systems

### Multiplayer Example
`multiplayer_example.py` - Client-server networking example with lobby system

## Running Examples

Make sure you have the required dependencies installed:

```bash
pip install -r ../requirements.txt
```

Then run any example:

```bash
python advanced_demo.py
python multiplayer_example.py
```

## C++ Examples

C++ examples can be found in the `src/` directory and are built using CMake.
